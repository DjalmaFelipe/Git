Como consultar a versão do Git
git --version

Como configurar username e email no GIT
git config --global user.name "Felipe"
git config --global user.email "DjalmaFelipe@outlook.com"

Como iniciar um novo repositório no Git
git init

Como verificar a situação do arquivo
git status

Nota:

TRACKED --> ocorre quando o arquivo está disponível no github
UNTRACKED --> essa condição se aplica quando o arquivo está disponível apenas localmente no seu dispositivo pessoal

Adiciona ao ambiente de testes e preparação (STAGE)
git add

Adicionando vários arquivos ao STAGE (escolher uma das quatro linhas de comando abaixo)
git add --all
git add -a
git add *
git add .

O commit serve como um save point, ou seja a versão finalmente é adicionada ao repositório, o commit deve ser acompanhado de uma mensagem descritiva
git commit -m "neste trecho deve ser inserida uma informação para descrever a construção ou ajuste realizado"

Como trazer um arquivo já existente no github para seu dispositivo
git clone

O comando deve ser seguido pelo link do repositório disponibilizado pelo github
Ex: git clone https://meurepositorio.git

Exibe o status com descrições reduzidas
git status --short
M index.html

NOTA: Os sinalizadores de status curtos são:

?? - Arquivos não rastreados
A - Arquivos adicionados ao stage/palco
M - Arquivos modificados
D - Arquivos excluídos

Como enviar arquivo diretamente, ignorando o envio ao STAGE
git commit -a -m "Updated index.html with a new line"

Como exibir histórico de modificações do repositório
git log

Como Associar repositório do github ao git
git remote add origin

O comando deve ser seguido pelo link do repositório disponibilizado pelo github
Ex: git clone https://meurepositorio.git

É possível consultar uma ajuda no própio git com os comandos:
git (comando) -help ou git help --all (este exibirá uma lista muito longa)
